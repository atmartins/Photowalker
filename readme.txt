Unit Testing
============
1. Install jasmine-node
	a. sudo npm install jasmine-node -g
2. Run tests:
	a. from command line at project root:
		i. jasmine-node spec


Dependencies
============
Note: Node dependencies are handled by an old-fashioned "npm install"
Note: See /Vagrant/bootstrap.sh for more dependencies (possibly).
1. Install exiftool
	a. apt-get install libimage-exiftool-perl
2. (recommended) Install usbmount if headless
	a. apt-get install usbmount
